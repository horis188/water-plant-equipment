# 供水厂设备管理平台 - 登录页面规格书

## 1. Concept & Vision

**概念：** 供水厂设备管理平台的登录页面，融合工业严谨感与水系流动美学。视觉语言以"水"为核心——流动、纯净、源源不断，传递供水企业守护民生基础设施的专业与可靠。

**情感定位：** 专业、可信赖、现代，同时带有水系独特的柔和流动感。

---

## 2. Design Language

### 色彩系统

| 用途 | 色值 | 说明 |
|-----|------|------|
| Primary | `#1E6BB8` | 深水蓝 - 主色调，稳重可靠 |
| Primary Light | `#4A9BD9` | 浅水蓝 - 悬停/强调 |
| Primary Dark | `#0D4A7A` | 深渊蓝 - 按钮按下/标题 |
| Secondary | `#2DD4BF` | 青绿色 - 辅助强调，活力感 |
| Accent | `#06B6D4` | 亮青色 - 亮点/交互反馈 |
| Background | `#F0F7FF` | 极浅蓝 - 页面背景 |
| Surface | `#FFFFFF` | 纯白 - 卡片/表单背景 |
| Text Primary | `#1A2A3A` | 深蓝灰 - 主要文字 |
| Text Secondary | `#5A7A8A` | 中灰蓝 - 次要文字 |
| Error | `#EF4444` | 错误提示 |
| Success | `#10B981` | 成功提示 |

### 字体

- **主字体**：`"Noto Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif`
- **标题字重**：700
- **正文字重**：400
- **字号体系**：
  - 大标题：32px
  - 副标题：20px
  - 正文：16px
  - 小字：14px
  - 验证码：24px

### 动效哲学

- **水流动效**：背景使用缓慢的波纹/水滴动画，营造水的流动感
- **表单交互**：输入框聚焦时边框产生涟漪效果
- **按钮反馈**：hover 时轻微"漂浮"感，click 时有"水波"扩散
- **过渡时长**：300-500ms，`ease-out` 为主

### 装饰元素

- **水流波纹**：底部渐变波纹背景
- **水滴粒子**：随机漂浮的水滴点缀
- **管道图标**：Login 图标使用抽象的管道/阀门元素
- **水循环图形**：Logo 区域使用循环水系抽象图形

### 视觉层级

1. **背景层**：淡蓝色渐变 + 动态波纹 + 漂浮水滴
2. **Logo 层**：平台名称 + 水循环抽象图形
3. **表单层**：白色半透明卡片，微微悬浮感
4. **装饰层**：底部水流/管道图形作为页脚

### 响应式策略

- **桌面（>1024px）**：表单居中，卡片宽度 420px
- **平板（768-1024px）**：表单宽度 85%，最大 420px
- **手机（<768px）**：表单宽度 95%，内边距减小

## 4. Features & Interactions

### 登录表单

#### 账号输入
- 图标：👤 人物图标
- placeholder：`请输入账号`
- 验证：非空，最小 4 字符
- 错误提示：账号格式不正确

#### 密码输入
- 图标：🔒 锁图标
- placeholder：`请输入密码`
- 显示/隐藏切换（眼睛图标）
- 验证：非空，最小 6 字符
- 错误提示：密码格式不正确

#### 验证码输入
- 4 位数字/字母混合
- placeholder：`请输入验证码`
- 验证：非空，4 位
- 刷新机制：点击验证码图片刷新
- 错误提示：验证码不正确

### 交互行为

| 元素 | Hover | Focus | Error | Disabled |
|-----|-------|-------|-------|----------|
| 输入框 | 边框变浅蓝 | 边框主色+阴影 | 边框红色+抖动动画 | 灰色背景 |
| 登录按钮 | 漂浮+光晕 | 深色+阴影 | - | 灰色不可点击 |
| 验证码图 | 轻微放大1.05x | 边框高亮 | - | 灰色 |

### 登录流程

1. 用户填写表单
2. 前端验证格式
3. 点击登录 → 按钮显示 loading
4. 调用登录接口
5. 成功 → 跳转首页
6. 失败 → 显示错误提示，输入框抖动

### 错误处理

- **网络错误**：`网络连接失败，请检查网络`
- **账号错误**：`账号或密码错误`
- **验证码错误**：`验证码错误，请重新输入`（清空验证码输入框）
- **账号冻结**：`账号已被冻结，请联系管理员`

## 5. Component Inventory

### LoginCard
白色半透明卡片，圆角 16px，内边距 40px，背景 `rgba(255,255,255,0.95)`，阴影 `0 8px 32px rgba(30,107,184,0.15)`

### WaterInput
- 高度：52px
- 圆角：8px
- 边框：1px `#D1E3F0`
- 聚焦：边框 2px `#1E6BB8`，box-shadow `0 0 0 3px rgba(30,107,184,0.15)`
- 左侧图标：24px，灰色

### WaterButton
- 高度：52px
- 圆角：8px
- 主色背景渐变：`linear-gradient(135deg, #1E6BB8 0%, #4A9BD9 100%)`
- 字体：16px，白色，字重 600
- Hover：向上移动 2px，阴影加深
- Active：向下移动 1px，阴影减小
- Loading：显示旋转水滴图标

### VerifyCode
- 尺寸：120x52px
- 背景：淡蓝色渐变 + 噪点纹理
- 字体：24px，字母数字混合，斜体
- 点击刷新：旋转动画

### RememberCheckbox
- 自定义样式：方形，圆角 4px
- 选中：主色背景 + 白色勾
- Label：14px，灰色文字

### BackgroundEffects
- **波纹**：底部 3 层渐变波纹，CSS animation 缓慢上下浮动
- **水滴**：随机位置的水滴粒子，缓慢下落动画
- **光晕**：右上角和左下角的光晕渐变

## 6. Technical Approach

### 技术栈
- **框架**：Vue 3 + Composition API (`<script setup>`)
- **构建**：Vite
- **语言**：TypeScript
- **样式**：CSS Variables + Scoped CSS（无需 Tailwind）
- **图标**：内联 SVG（自定义水系图标）

### 目录结构

```
src/
├── views/
│   └── LoginView.vue      # 登录页主视图
├── components/
│   ├── LoginCard.vue     # 登录卡片组件
│   ├── WaterInput.vue    # 水系风格输入框
│   ├── WaterButton.vue    # 水系风格按钮
│   ├── VerifyCode.vue     # 验证码组件
│   ├── WaterBackground.vue # 动态水背景
│   └── icons/
│       └── WaterIcons.vue  # 水系 SVG 图标
├── composables/
│   └── useLogin.ts       # 登录逻辑
├── services/
│   └── auth.ts           # 认证接口
├── styles/
│   └── variables.css     # CSS 变量
├── router/
│   └── index.ts          # 路由配置
└── App.vue
```

### 验证码生成
- 前端生成：Canvas 绘制或 SVG 随机字符
- 字符集：`A-Z0-9`
- 干扰线：2-3 条半透明直线
- 噪点：随机小点

### 登录接口（Mock）

```typescript
// POST /api/login
interface LoginRequest {
  username: string

interface LoginResponse {
  code: number
```

## 7. 设计参考

### 视觉关键词
- 水流、波浪、涟漪
- 管道、水龙头、水滴
- 蓝色渐变、透明质感
- 工业严谨 + 水的柔和

### 优秀案例参考
- 水利局/水务公司官网的蓝色系统
- dribbble 上的 water/wave 主题 UI
- 现代 SaaS 登录页的简洁风格

### 禁止出现
- ❌ 红色为主色调（与水厂主题不符）
- ❌ 过于卡通化的图形
- ❌ 复杂的三维效果（保持现代简洁）
